Proyecto Todolist -- Prueba de concepto -- SEREMPRE
Versión inicial (21/01/2020) Ramiro Palmiery

Descripción general: 

Proyecto desarrollado implementando una capa de servicios REST elaborados con Spring Framework.
El API se despliega en un contenedor SpringBoot. Por defecto se ejecuta en el puerto 8080. 
Para la construcción del API fue generado un proyecto maven y se utilizó el IDE Eclipse.

El proyecto web está desarrollado utilizando Angular 8 y Bootstrap. Desarrollado con el IDE Visual Studio Code.

La persistencia de datos se encuentra en MySql. Para crear la base de datos se debe utilizar el script adjunto 20200121TodolistDB_start.sql.

Pasos de instalación del ambiente de desarrollo:
1. Instalar JDK 8 o superior.
2. Instalar Mysql.
3. Correr el Script de arranque (20200121TodolistDB_start.sql)
4. Instalar IDE Eclipse.
5. Importar el proyecto en Eclipse (Opción "Importar proyecto maven" e ir al path donde se almacena la carpeta todolist)
6. Instalar IDE Visual Studio Code.
7. Abrir la carpeta del proyecto web todolist-rest
